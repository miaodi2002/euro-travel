### ファイル命名規則

**航空券**：`月:日 出発地→経由地→到着地[_姓].pdf`
- 例：`7:8 HND→DXB→MXP_Miao.pdf` / `7:30 FRA→NRT_Miao.pdf`
- 2 名同一 PDF：サフィックスなし
- 個別 PDF：`_Miao.pdf` / `_Sakoda.pdf`

**列車券**：`月:日 出発地 → 到着地.pdf`（または `.txt`）
- 例：`7:11 Milano→Venezia.pdf` / `7:28 Lauterbrunnen → Frankfurt.txt`

**ホテル**：`月:日-月:日 都市 ホテルキーワード.pdf`
- 例：`7:9-7:11 Milano Casa Brera.pdf` / `7:20-7:25 Barcelona EDITION.pdf`
- 都市は英語 / ローマ字（Milano / Venezia / Firenze / Roma / Santorini / Barcelona / Lauterbrunnen / Frankfurt）
- ホテルキーワードは識別できれば十分（全名不要、Casa Brera / W Florence / EDITION）
- 同一ホテルで複数区間：`_1` `_2` サフィックス

**レストラン**：`月:日 都市 レストランキーワード.pdf`
- 例：`7:17 Santorini Lycabettus Andronis.pdf` / `7:12 Venezia Osteria Testiere.pdf`
- 同日複数軒：時刻サフィックス、例 `7:18 Santorini Sunset Ammoudi 13時.pdf`
- レストランキーワードは識別できれば十分
