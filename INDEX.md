### 命名约定

**机票**：`月:日 起点→中转→终点[_姓氏].pdf`
- 示例：`7:8 HND→DXB→MXP_Miao.pdf` / `7:30 FRA→NRT_Miao.pdf`
- 同 PDF 两人：不加后缀
- 分人单 PDF：`_Miao.pdf` / `_Sakoda.pdf`

**火车票**：`月:日 起点 → 终点.pdf`（或 `.txt`）
- 示例：`7:11 Milano→Venezia.pdf` / `7:28 Lauterbrunnen → Frankfurt.txt`

**酒店**：`月:日-月:日 城市 物业关键词.pdf`
- 示例：`7:9-7:11 Milano Casa Brera.pdf` / `7:20-7:25 Barcelona EDITION.pdf`
- 城市用英文 / 罗马字（Milano / Venezia / Firenze / Roma / Santorini / Barcelona / Lauterbrunnen / Frankfurt）
- 物业关键词够区分即可（不必全名，Casa Brera / W Florence / EDITION）
- 同物业多段：`_1` `_2` 后缀

**餐厅**：`月:日 城市 餐厅关键词.pdf`
- 示例：`7:17 Santorini Lycabettus Andronis.pdf` / `7:12 Venezia Osteria Testiere.pdf`
- 同一天多家：加时刻后缀，如 `7:18 Santorini Sunset Ammoudi 13时.pdf`
- 餐厅关键词够区分即可（不必全名）

**景点票**：`月:日 景点关键词.pdf`（放 `Attractions/`）
- 示例：`7:26 Gelmerbahn.pdf` / `7:27 Jungfraujoch.pdf`
- 同日多个：加关键词/时刻区分
